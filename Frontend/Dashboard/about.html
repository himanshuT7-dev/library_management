About Our System

A modern, database-driven Library Management System built with Java Spring Boot and JavaScript.

📘 About the Project

This project is a modern Library Management System designed as a Bachelor's degree final year presentation. It demonstrates secure user authentication, a RESTful API backend architecture, and dynamic front-end data binding.


Key Technologies Used:

  • 💻 Frontend: HTML5, Vanilla CSS3, Javascript (ES6+, Fetch API)
  • ☕ Backend: Java Spring Boot, Spring Security, JPA Hibernate
  • 🗄️ Database: MySQL
